# Trade Control

The Systems Engineering principles and history behind [Trade Control](https://github.com/tradecontrol) from the perspective of its creator. 

## History

Trade Control was not originally an open source app, so for completeness this repository contains the code from its original release on 19 Mar 08 to the GitHub upload on 24 Sept 19. 

## Code 

The T-SQL script archive for Trade Control is presented in a Visual Studio Sql project called _scripts.sqlproj_ in the _src/tc-history.sol_ solution.
